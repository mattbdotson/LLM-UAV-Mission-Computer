L0-PER-08Geo-location accuracy meets bound sufficient for reasoning.The geo-projector in the edge Perception service produces the accuracy; the bound value is TBD, derived at L1.The edge Geo-Projector produces the accuracy; the bound value is TBD, derived at L1.
